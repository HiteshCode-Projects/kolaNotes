Display None is a Property that can be applied to an element/Tag to hide it from the page.

Hello Display None

Display Block is a Property that can be applied to an element/Tag to make it a block element.
It will take the full width of the page and start on a new line.

Hello Display Block

Display Inline is a Property that can be applied to an element/Tag to make it an inline element.
It will only take the width of its content and will not start on a new line.

Hello Display Inline
